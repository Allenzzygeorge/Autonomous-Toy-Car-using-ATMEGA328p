## Tech Stack

- **MCU:** ATMEGA328p (AVR)
- **Motor driver:** L298N (or better: TB6612 / DRV8833)
- **Sensors:** Ultrasonic HC-SR04, IR line sensors (optional)
- **Tools & Software:** avr-gcc, avrdude, PlatformIO / Arduino IDE, C
- **Others:** LiPo / battery management, breadboard / PCB

---

## Skills & Topics Covered

- Embedded C and AVR microcontroller programming  
- PWM motor control and H-bridge drivers  
- Ultrasonic distance sensing & obstacle avoidance  
- Serial bootloading & flashing with avrdude  
- PCB prototyping and wiring  
- Basic electronics: voltage regulation & motor power management
